ce projet a pour objet de bâtir un site internet, destiné à permettre à tout à chacun de poster un article,
et de recevoir des commentaires pour chaque article.
le caractère responsive, ainsi que le slider sont établis uniquement à partir du CSS.
pour cela:
site : définir une largeur max avec max-width pour site
img : pour la classe image, définir qu'elles occupent 100% de la zone allouée avec width: 100%
slider: pour gérer un slider, on aligne les images les unes à côté des autres avec display: flex puis,
on utilise la propriété overflow: auto.
on ajoute enfin l animation.
